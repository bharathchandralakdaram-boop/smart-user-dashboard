# Smart User Dashboard

A modern React dashboard application built using Vite, Tailwind CSS, and API Service Layer architecture. The project integrates Users, Products, and Posts data from DummyJSON APIs.

---

## 🚀 Features

- Users API Integration
- Products API Integration
- Posts API Integration
- React Hooks
- State Management
- Responsive Dashboard UI
- Tailwind CSS Styling
- Service Layer Architecture
- React Router Navigation

---

## 🛠 Technologies Used

- React
- Vite
- Tailwind CSS
- React Router DOM
- DummyJSON API
